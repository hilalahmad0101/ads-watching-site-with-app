# AdReward Pro - Rewards & Ads Management Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Flutter](https://img.shields.io/badge/Flutter-v3.0+-blue.svg)](https://flutter.dev)
[![HTML5](https://img.shields.io/badge/HTML5-Modern-orange.svg)](https://w3.org)

**AdReward Pro** is a comprehensive rewards and advertising management ecosystem, featuring a robust Flutter mobile application and a powerful HTML-based admin dashboard. This platform enables users to earn rewards by watching advertisements, completing challenges, and participating in a referral network, while providing administrators with elite tools for platform oversight.

## 🚀 Key Features

- **Dynamic Reward System**: Earn credits/points for engagement.
- **Advanced Admin Dashboard**: Premium, data-rich interface for total control.
- **Ad Management**: Seamlessly manage advertisement streams.
- **Secure Financials**: Robust withdrawal management and transaction history.
- **Referral Network**: Multi-tier referral tracking for organic growth.
- **Cross-Platform Experience**: High-fidelity Flutter UI for mobile and responsive HTML templates.

## 🛠️ Project Structure

- **/stitch_ad_reward_flutter**: The core mobile application codebase (Flutter).
- **/admin_code**: Specialized HTML templates and scripts for the admin panel.
- **/html_code**: Legacy or alternative HTML versions of the mobile screens.
- **/screenshot**: High-resolution design assets and application previews.

## ⚙️ Installation

### Mobile App (Flutter)
1. Ensure you have the Flutter SDK installed.
2. Navigate to `/stitch_ad_reward_flutter`.
3. Run `flutter pub get`.
4. Launch with `flutter run`.

### Admin Dashboard (HTML)
1. Navigate to `/admin_code`.
2. Open `admin_dashboard_premium_dark.html` in any modern web browser.
